# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this toolkit, **please do not open a public GitHub issue.**

Report vulnerabilities privately to:
**Email:** pulasthibatuwita9@gmail.com
**Subject:** `[SECURITY] cmmc-enclave-toolkit`

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes (optional)

We will acknowledge receipt within 72 hours and work to release a fix as quickly as possible.

## Scope

This policy covers the code in this repository. It does not cover:
- Third-party dependencies (report to those projects directly)
- The Docker base images (report to Docker Hub / upstream)
- WireGuard (report to the WireGuard project)
